Gratitude

Style guide

The single source of truth for Phase 3. A warm, hand-kept journal whose signature is seasonality — the whole canvas shifts through six UK mini-seasons. Handwriting (Caveat) for the voice, a warm humanist sans (Figtree) for the words, fixed per-category colours, hand-drawn crayon entry cards, and a painterly watercolour hero per season. Light only by design.

Phase 2 · style guide · light-only · fonts Caveat + Figtree (OFL)

Foundations

Type

Two families, both OFL / self-hostable. Caveat carries the personality — wordmark, day dates, category headings, timestamps. Figtree does the reading work — entry text, labels, buttons. Handwriting stays precious to structure; the words people write are always the clean sans.

Foundations

Six seasons

The app's signature. The accent (date + FAB + buttons), the radial washes, and the hero watercolour all rotate by mini-season; paper stays warm cream throughout — even Winter — so it never reads as “the blue app”. Each card shows its hero and a swatch row: accent · accent-deep · paper · card.

Components

Parts

The hand-drawn pieces, shown on Early Summer paper. The entry card's wobble comes from an SVG turbulence/displacement filter on a doubled border (three seeds cycled so neighbours differ); only the outline is filtered, so text stays crisp. The selected chip reuses the same ink at chip scale.

The day, six ways

The home screen — one day, entries grouped by category, chronological within a group. Same structure every month; only the season changes the accent, washes and hero. Floating + FAB bottom-right for one-handed reach.
